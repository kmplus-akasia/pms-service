## Catatan Klasifikasi

1. **Planning**: User stories terkait drafting, submit, approval KPI/KAI, cascading, dan konfigurasi target
2. **Monitoring**: User stories terkait input realisasi, approve realisasi, melihat achievement/score
3. **Both**: Fitur yang tersedia di kedua periode dengan kemungkinan tampilan atau behavior berbeda (dashboard, view detail, fitur pencarian)
